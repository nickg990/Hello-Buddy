# Architecture Decision Records

Store architecture decisions for the Admin solution in this folder.

## Naming

Use the Nygard style with a numeric prefix:

- `0001-short-title.md`
- `0002-short-title.md`

## Minimum content

- Context
- Decision
- Consequences

Link each new ADR in `Planning/Admin Decision Log.md` in the same change train.
