# HelloBuddy.Ui

ASP.NET Core MVC UI for practitioners, calling the Admin API through the typed client service layer.

## Run

```powershell
dotnet run --project src/HelloBuddy.Ui/HelloBuddy.Ui.csproj --launch-profile http
```

## Test

```powershell
dotnet test tests/HelloBuddy.Ui.Tests/HelloBuddy.Ui.Tests.csproj
```
